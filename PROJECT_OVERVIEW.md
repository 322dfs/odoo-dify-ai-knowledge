# Odoo + Dify 企业内网 AI 智能知识库

## 一、项目背景

### 1.1 公司现状

- 半导体初创公司
- 员工不爱写文档，技术经验容易流失
- 新人上手慢，重复问题多
- 缺乏统一的知识管理平台

### 1.2 核心痛点

| 痛点     | 影响            |
| ------ | ------------- |
| 文档记录缺失 | 技术经验随人员流动流失   |
| 知识分散   | 部署手册、故障案例散落各处 |
| 重复问题多  | 老员工反复解答相同问题   |
| 新人培训慢  | 缺乏系统化的知识传承    |

### 1.3 项目目标

构建**私有化 AI 知识库**，实现：

1. 统一存储公司文档、故障案例、技术规范
2. AI 智能问答，快速检索相关知识
3. 全内网部署，数据安全合规

## 三、功能模块

### 3.1 Odoo 知识库模块

| 功能   | 说明                    |
| ---- | --------------------- |
| 文档管理 | 新建、编辑、查看、删除文档         |
| 分类管理 | 按类型分类（部署手册、故障案例、技术规范） |
| 搜索功能 | 关键词搜索文档               |
| 权限控制 | 基于角色的访问控制             |

### 3.2 Dify AI 问答模块

| 功能     | 说明              |
| ------ | --------------- |
| 智能问答   | 输入问题，AI 基于知识库回答 |
| RAG 检索 | 向量检索相关文档片段      |
| 来源追溯   | 显示答案来源的文档       |
| 对话历史   | 保存对话记录          |

### 3.3 数据同步模块

| 功能   | 说明                    |
| ---- | --------------------- |
| 定时同步 | 定时从 Odoo 拉取文档同步到 Dify |
| 增量更新 | 只同步新增/修改的文档           |
| 状态监控 | 同步状态日志记录              |
